Hexo安装

Ranger

准备

Hexo安装前，请确保你的电脑上已经安装了Node.js和Git。网上已有很多教程，且安装过程很简单，在此只简单介绍一下。

安装Git

  • Windows系统: 点击链接git，下载安装。
  • Mac: 点击链接git，下载安装。如果已经安装Homebrew，的话，可以直接在终端输入brew install git。
  • Linux (Ubuntu, Debian): 终端输入sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS):终端输入 sudo yum install git-core

安装Node.js

点击链接Node.js，下载安装。Mac用户也可通过brew install node安装。

安装Hexo

上述条件满足后，只需在终端输入
npm install -g hexo-cli
即可安装hexo。
