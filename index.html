Hexo安装

Ranger

准备

Hexo安装前，请确保你的电脑上已经安装了Node.js和Git。网上已有很多教程，且安装过程很简单，在此只简单介绍一下。

安装Git

  • Windows系统: 点击链接git，下载安装。
  • Mac: 点击链接git，下载安装。如果已经安装Homebrew，的话，可以直接在终端输入brew install git。
  • Linux (Ubuntu, Debian): 终端输入sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS):终端输入 sudo yum install git-core

安装Node.js

点击链接Node.js，下载安装。Mac用户也可通过brew install node安装。

安装Hexo

上述条件满足后，只需在终端输入
npm install -g hexo-cli
即可安装hexo。

本想继续更新，主要是将Hexo官网中的英文说明文档翻译成中文，再添加一些自己使用中的一下经验。偶然发现，官网有中文的说明文档:(
大家自行参考吧：<https://hexo.io/zh-cn/docs/index.html>

新篇

Ranger

一直想写点东西，拯救一下自己日益衰退的文字功力。犹记初中后半段到高中，那些还有语文课的日子，我考试的时候作文从未写完过。大学选择了理科，从此远离作文，开始了与公式推导朝夕相伴的日子。

步入研究生阶段，面对的问题更与文字无关，而是编程。公式推导很难求解庞大复杂的系统，编程模拟变得越来越重要。从最开始的Matlab、Fortran，到后来的Python，大多依靠自己不断摸索，或者研究别人的程序学习的。内心对编程还是挺感兴趣的，只是畏惧于它的繁杂。从没想过要系统地学习编程，只是针对当前面临的问题而寻求相应的编程求解方法。如此接触了好多编程语言和方法，但都是浅尝辄止，偶尔用过几次，后面又忘记了。由于编程的目的性太强，只为求解问题，而忽略了一些编程的规范，自己写的程序的可读性很差。而我本身又是一个粗心大意的人，程序往往写了多个版本，到最后连自己都分不清哪个是最新版了。

最近刚接触GitHub，这或许可以促使我写程序更加条理一些。偶然发现Hexo这个程序，索性就在GitHub上建一个主页，无意分享，只为记录。一是记录自己学习编程的一些心得，一是记录一些日常点滴，或许还有一些专业课的知识。

写了两百多字，已无从下笔。回看一遍，更是逻辑混乱，毫无主线。嗯，这就是我面临地问题，这也是我建立这个博客最想解决的问题。
